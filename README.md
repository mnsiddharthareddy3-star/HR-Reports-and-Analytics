# HR Reports / Analytics System (C Project)

A simple **HR Management and Analytics System** built using **C programming**.  
This project allows users to add, display, search, update, delete employees and generate salary-based reports.

---

## üìå Features

### 1. Add Employee  
Allows adding new employee details:
- ID  
- Name  
- Department  
- Salary  
- Age  

### 2. Display All Employees  
Shows the complete list of employees stored in the system.

### 3. Search Employee by ID  
Finds and displays employee details based on their unique ID.

### 4. Average Salary Report  
Calculates and displays the average salary of all employees.

### 5. Highest Salary Report  
Finds the employee with the highest salary and displays their information.

### 6. Salary Distribution  
Categorizes employees based on their salary range:
- Below ‚Çπ20,000  
- ‚Çπ20,000 to ‚Çπ50,000  
- Above ‚Çπ50,000  

### 7. Update Employee  
Modify details of an existing employee.

### 8. Delete Employee  
Remove an employee from the list by ID.

### 9. Exit  
Closes the application.
